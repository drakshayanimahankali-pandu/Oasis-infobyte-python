# Oasis Infobyte Python Projects

This repository contains Python projects developed during the Oasis Infobyte Internship Program.

## Projects Included

### 1. Basic Weather App

A weather application developed using Python Tkinter GUI and OpenWeather API.

#### Features

* Real-time weather updates
* Temperature display
* Humidity information
* Wind speed details
* Weather condition display

#### Technologies Used

* Python
* Tkinter
* Requests Library
* OpenWeather API

---

### 2. Random Password Generator

A Python application that generates secure random passwords.

#### Features

* Random strong password generation
* User-friendly interface
* Custom password length support

#### Technologies Used

* Python
* Random Module
* Tkinter

---

### 3. Voice Assistant

A simple AI Voice Assistant developed using Python.

#### Features

* Voice commands
* Speech recognition
* Text-to-speech output
* Web search support

#### Technologies Used

* Python
* pyttsx3
* SpeechRecognition
* Wikipedia API

---

### 4. Browser Based Chat Application

A client-server based chat application developed using Python socket programming.

#### Features

* Real-time chatting
* Multiple client communication
* Server-client architecture

#### Technologies Used

* Python
* Socket Programming

---

## How to Run

### Install Required Libraries

pip install requests pyttsx3 SpeechRecognition wikipedia

### Run Projects

#### Weather App

python basicweatherapp.py

#### Password Generator

python RandomPasswordGenerator.py

#### Voice Assistant

python VoiceAssistant.py

#### Chat Application Server

python browserbasedchatapplicationserver.py

#### Chat Application Client

python browserbasedchatapplicationclient.py
